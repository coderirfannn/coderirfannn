### Engineering Principles

- **Ship, then refine.** A working deployed version beats a perfect local one.
- **Own the full stack.** Frontend, backend, database, and deployment should all be understood, not outsourced to a tutorial.
- **Design for failure.** Auth flows, API calls, and background jobs should fail predictably, not silently.
- **Measure before claiming.** No feature is "optimized" without a before/after number to show for it.

<summary><b>How to enable the snake animation above</b></summary>
<br/>

This requires a GitHub Action in your profile repo (`coderirfannn/coderirfannn`). Create `.github/workflows/snake.yml`:

```yaml
name: generate snake animation
on:
  schedule:
    - cron: "0 */6 * * *"
  workflow_dispatch: {}
  push:
    branches: [ main ]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: coderirfannn
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Once it runs once, the badges above will populate automatically.
